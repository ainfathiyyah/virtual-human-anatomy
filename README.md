# Virtual Human Anatomy

Virtual Human Anatomy is a Final Year Project (FYP) developed using Virtual Reality technology to enhance anatomy learning experiences.

## Project Overview

This project provides an immersive learning environment where students can explore:

- Skeletal System
- Muscular System
- Human Organs
- Interactive Learning Activities
- Educational Quizzes

## Features

- Interactive 3D anatomy models
- Virtual Reality learning environment
- Educational infographics
- Anatomy stickers
- Learning assessment quizzes
- User-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Unity 3D
- Virtual Reality (VR)
